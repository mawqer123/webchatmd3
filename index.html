关于资源分享站 MD2

本站是一个基于 GitHub API 的资源分享平台，所有资源均由社区用户投稿分享。

资源存储于 GitHub 仓库，通过 GitHub API 动态加载。本站不存储任何文件，仅提供索引和访问服务。

主要功能：

  • 按分类浏览资源文件
  • 关键词搜索文件
  • 在线预览文件内容
  • 用户投稿分享

技术栈： HTML5, CSS3, JavaScript, GitHub API
